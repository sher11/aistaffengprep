Interview Framework: RESHADE

Use this framework to structure your 45-60 minute system design interview.

R - Requirements

5-8 min: Clarify functional & non-functional requirements. Ask about scale, users, and constraints.

E - Estimation

3-5 min: Calculate QPS, storage, bandwidth. Use back-of-envelope math.

S - Storage Schema

3-5 min: Design data model, choose SQL vs NoSQL, define key entities.

H - High-Level Design

10-15 min: Draw component architecture with load balancers, services, databases.

A - API Design

3-5 min: Define REST/GraphQL endpoints, request/response formats.

D - Deep Dive

15-20 min: Detail critical components, discuss trade-offs, handle edge cases.
