# Coding Rounds - Resources Inventory

## Foundational Resources

### 1. MIT Algorithm Lectures
- **Link**: https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/video_galleries/lecture-videos/
- **Summary**: Comprehensive course covering fundamental algorithms and data structures from MIT's Introduction to Algorithms course.
- **Key Takeaways**:
  - Rigorous analysis of algorithm complexity
  - Proof techniques for correctness
  - Foundational data structures (heaps, BSTs, hash tables)
  - Graph algorithms and dynamic programming
- **Difficulty Level**: Intermediate
- **Estimated Time**: 40+ hours for complete coverage

### 2. LeetCode Interview Crash Course
- **Link**: https://leetcode.com/explore/featured/card/leetcodes-interview-crash-course-data-structures-and-algorithms/
- **Summary**: Quick refresher on essential data structures and algorithms optimized for interview preparation.
- **Key Takeaways**:
  - Pattern-based problem solving
  - Time-efficient review of core concepts
  - Practice problems aligned with each topic
- **Difficulty Level**: Beginner to Intermediate
- **Estimated Time**: 15-20 hours

## Practice Problem Sets

### 3. Blind75 / Grind75
- **Link**: https://leetcode.com/problem-list/rab78cw1/
- **Summary**: Curated list of 75 most common interview questions covering breadth of topics.
- **Key Takeaways**:
  - Essential problems every candidate should know
  - Good breadth coverage across all major topics
  - Efficient use of limited preparation time
- **Difficulty Level**: Easy to Hard (mixed)
- **Estimated Time**: 50-75 hours (aiming for 30-40 min per problem)

### 4. NeetCode150
- **Link**: https://leetcode.com/problem-list/pl0tvgui/
- **Summary**: Extended version of Blind75 with additional problems for deeper coverage.
- **Key Takeaways**:
  - More comprehensive topic coverage
  - Organized by pattern/category
  - Video explanations available at neetcode.io
- **Difficulty Level**: Easy to Hard (mixed)
- **Estimated Time**: 100-150 hours

## Non-LeetCode Style Resources

### 5. Real-World Coding Challenges
- **Source**: StaffEngPrep, company-specific guides
- **Summary**: Practical coding problems that mirror actual work scenarios.
- **Key Problem Types**:

| Problem | Description | Skills Tested |
|---------|-------------|---------------|
| LRU Cache | Implement with O(1) operations | Data structure design, HashMap + LinkedList |
| Log Parser | Parse and analyze log files | String processing, data aggregation |
| SQL Engine | Implement basic SQL operations | Parsing, data structures, query execution |
| Web Crawler | Crawl pages with constraints | BFS/DFS, concurrency, rate limiting |
| Read/Write Locks | Implement synchronization primitive | Concurrency, thread safety |

- **Difficulty Level**: Intermediate to Advanced
- **Estimated Time**: 2-4 hours per problem
