Generative AI Engineering

Master LLMs, RAG, Agents, and Production AI Systems. Become a strong AI Engineer who can design, build, and ship LLM systems.

Course Progress 0% Complete

Course Philosophy

  • Code-first, intuition-first - Math only when it builds understanding
  • Real-world engineering - Every concept connects to production decisions
  • WHY before HOW - Understand the reasoning, not just the implementation
  • Active learning - Spaced repetition, mini challenges, and hands-on projects

Who This Course Is For

  • Senior software engineers / backend engineers
  • Comfortable with Python, basic ML, system design
  • Goal: Become a strong AI Engineer who can design, build, and ship LLM systems

Prerequisites

Required Knowledge

  • Python proficiency - Classes, decorators, async/await, type hints
  • Basic ML concepts - What is a model, training vs inference, overfitting
  • System design basics - APIs, databases, caching, scalability concepts

Helpful but Not Required

  • Linear algebra (matrices, vectors) - We'll cover what you need
  • PyTorch or TensorFlow experience - We'll use PyTorch
  • Prior NLP experience - Helpful but not necessary

Additional Resources

Recommended Reading

  • Deep Learning Book - Goodfellow, Bengio, Courville
  • Attention Is All You Need - The Transformer paper
  • The Illustrated Transformer - Jay Alammar
  • Andrej Karpathy's Blog - Neural networks explained

Tools We'll Use

  • PyTorch - Primary deep learning framework
  • Hugging Face Transformers - Pre-trained models and tokenizers
  • LangChain / LangGraph - Agent orchestration
  • Vector databases - Pinecone, Chroma, Weaviate
  • OpenAI / Anthropic APIs - Commercial LLM access
