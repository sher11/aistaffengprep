CodeSignal Assessment

Assessment Structure ▼
  • Duration: 70-90 minutes
  • Format: 4 levels of increasing difficulty
  • Target: 520+ out of 600 to advance
Level Points Time Priority
Level 1~10010-15 minMust complete
Level 2~15015-20 minMust complete
Level 3~17520-25 minShould complete
Level 4~17520-25 minNice to have
Scoring Factors ▼
  • Correctness: Tests pass
  • Cleanliness: Formatting, naming
  • Modularity: Function decomposition
  • Efficiency: Reasonable complexity

Sample Problem: Inventory System

Level 1-2: Basic CRUD
- add_item(id, quantity)
- remove_item(id, quantity)
- get_quantity(id)
- search_by_prefix(prefix)

Level 3: Multi-user with capacity
- reserve_for_user(user_id, item_id, qty)
- set_capacity_limit(item_id, limit)

Level 4: Duplicate handling
- merge_duplicates()
- resolve_over_capacity()

Time Management Strategy

First Pass (Levels 1-2)

Complete quickly, don't over-engineer. These are foundation points.

Second Pass (Level 3)

Build on existing code. Add new methods carefully.

Final Push (Level 4)

Only if time allows. Don't break earlier levels.

Buffer Time

Keep 5-10 min for testing and debugging.
