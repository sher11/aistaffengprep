Weeks 1-4: Build Foundations

  • Start System Design Module 1-2 (Scalability, Databases)
  • Practice 3-5 coding problems daily
  • Read DDIA chapters 1-4

5-8

Weeks 5-8: Deepen Understanding

  • Complete Coding Modules 5-7 (DP, Backtracking, Heaps)
  • System Design Modules 3-7 (Distributed Systems, APIs)
  • Begin Company-Specific research
  • Start preparing 5-7 behavioral stories

9-12

Weeks 9-12: Interview Preparation

  • Complete Coding Module 8-9 (Non-LeetCode, Interview Simulation)
  • System Design Mock Interviews (2-3 per week)
  • Complete Company-Specific prep for target companies
  • Practice behavioral answers with recording
  • Schedule real mock interviews
Primary: Mock Interviews Secondary: Company Fit
